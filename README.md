fork本仓库，然后在`Settings`→`Security`→`Secrets`→`Actions`选项卡中，添加`New repository secret`：

- `COOKIE`为`COOKIE`
- `SEND_KEY`为`Server酱SEND_KEY`

默认每日UTC时间13时运行。
